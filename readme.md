# Email Checker

## Installation

```bash
# Manage deps
virtualenv venv
source venv/bin/activate

# Install all dependencies
pip install -r requirements.txt
```


## USAGE

### Test your own email
```bash
python email_checker.py your.email@gmail.com
```
